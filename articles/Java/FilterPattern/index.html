一. Filter原理

最近刚刚接触dubbo，瞬间被dubbo的简洁和扩展性圈粉了，用郭大爷的话来说就是又勾勾又丢丢，一想之美。代码算上测试一共短短12W行，没有像spring那么复杂的继承结构，但是对各种设计模式和架构设计理念的运用又极度的优雅。
