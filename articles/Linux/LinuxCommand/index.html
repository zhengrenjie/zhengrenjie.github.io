常用Linux指令

2019-03-17 Linux Linux Comments

df -h

查看文件系统的使用磁盘使用量。

-h —human 表示将结果以容易阅读的方式展现。

du -sh *

查看当前文件夹下所有文件占用磁盘大小。

fc-list :lang=zh

查看当前系统中的中文字体。

ps aux | grep XXX

查找XXX进程。

lspci | grep -i vga

查看当前系统中的显卡信息。

lsof -i:8080

查看当前8080端口占用，lsof: list open file。

find . -name “*.c”

查找当前目录下所有已”.c”结尾的文件。

/var/spool/cron/

Linux cron文件的保存位置，老是找不到 = =
